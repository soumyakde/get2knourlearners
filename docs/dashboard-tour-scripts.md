# Dashboard Tour Scripts — For Screen Recording

*Use with ScreenPal (or similar). Each numbered item is one beat: show this, say this. Times are per-item estimates for pacing while narrating — actual recording will vary with your speaking pace. Each tour stays under ScreenPal's 15-minute free-tier cap; the Teacher Dashboard tour is split into three shorter videos since it's the most complex.*

---

## Tour 1 — Student Dashboard
**Target length: ~3–4 minutes. One take.**

| # | Show | Say (roughly) | Est. time |
|---|---|---|---|
| 1 | Login screen | "Every learner gets a simple username and password — no email required." | 15s |
| 2 | Student Dashboard, full module list | "Here's the whole course, laid out in order. Green means done, blue means available now, gray means locked." | 30s |
| 3 | Point at a locked module | "Locked modules unlock automatically once the one before it is finished — no manual unlocking needed." | 20s |
| 4 | Click "Open" on the available module | "Let's open the current module." | 10s |
| 5 | Show the lesson content briefly | "Each module starts with a short lesson." | 20s |
| 6 | Scroll to the MCQ assessment | "Then a quick multiple-choice check for understanding." | 20s |
| 7 | Answer 2–3 questions, submit | "Answer, submit — scored instantly, no waiting." | 40s |
| 8 | Show the survey instrument | "Next, a brief survey about how engaged and motivated they felt." | 25s |
| 9 | Show the reflection question | "And a short written reflection, in their own words." | 20s |
| 10 | Submit, return to dashboard | "Submit, and back on the dashboard — that module is now marked complete, and the next one just unlocked." | 30s |

**Total: ~3.5 min**

## Tour 3 — Teacher Dashboard, Part 2: Advanced Analytics
**Target length: ~9–10 minutes.**

| # | Show | Say | Est. time |
|---|---|---|---|
| 1 | Inferential Statistics tab | "This is where it gets powerful — the platform runs the actual statistical tests, automatically." | 20s |
| 2 | Run a paired t-test / ANOVA | "Pick two time points, and it runs the test and explains the result in plain language." | 45s |
| 3 | Bland-Altman panel | "For comparing two measurement occasions — like pre- and post-course scores — it shows the limits of agreement, not just whether the average changed." | 40s |
| 4 | IRT Analysis tab | "Item Response Theory analysis — tells you which assessment questions are actually doing their job, item by item." | 40s |
| 5 | Construct reliability panel (CR/AVE) | "And full psychometric reliability reporting — the same rigor a published study would need." | 35s |
| 6 | LLM Analysis tab | "Now the qualitative side — reflections get coded automatically by AI against themes you define." | 30s |
| 7 | Show a coded theme with citations | "Every theme traces back to exactly which student said what — full transparency, no black box." | 35s |
| 8 | Competency Progression (CPI) tab | "And this ties it all together — one composite score per student, blending content knowledge with engagement and motivation." | 40s |
| 9 | Show a student's CPI band | "Each student lands in a performance band, with the reasoning behind it." | 30s |

**Total: ~5.5 min** *(room to slow down / add follow-up questions and answers)*

## Suggested recording order

1. Student Dashboard (simplest — good warm-up)
2. Teacher Dashboard Part 1 (Overview & Basic Stats)
3. Admin Dashboard
4. Teacher Dashboard Part 2 (Advanced Analytics) — most complex, save for when you're warmed up
5. Teacher Dashboard Part 3 (Reports) — short, good closer

## Before you record

- Use a **test/demo account** with realistic-looking but non-identifying sample data — never record with real student names or data visible.
- A quiet room and a decent headset mic matter more than the software — ScreenPal's built-in mic recording is fine for this.
- Do one unscripted practice run per tour first; the timings above are pacing guides, not a script to read verbatim.
